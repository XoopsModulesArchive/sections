Xoops Koivi Editor
by Samuels
The XOOPS Project

Xoops Koivi editor is a xoopsform class.
It allows to use a wysiwyg (what you see is what you get) text area instead a normal textfield.

Advantages:

1. It's made to fit with Xoops.
2. Fully customizable, (skins).
3. Crossbrowser Compatible.
4. It's POO.
5. XTHML compilant.
6. Unlimited instances on the same page.
7. Advanced wysiwyg html tools.

Browser Compatibility list:

Mozilla Firefox 0.8 and upper
Internet Explorer 5.5 and upper
Mozilla 1.3 and upper
Netscape 7.1 and upper

Tools:

Cut, Copy, Paste, Forecolor, Backcolor, Remove Word Format, Insert Symbols, Undo an Redo, Advanced Table tools, Xoops Imagemanager Integration, Xoops Code and Quote, Floating toolbar e.t.c
